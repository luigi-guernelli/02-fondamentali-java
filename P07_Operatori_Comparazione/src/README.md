# Classe Main (Operatori di Comparazione)
Questa classe dimostra operatori di comparazione, che confrontano valori e restituiscono  *boolean* **(ture/false)**.
Usati in condizioni e3s.(*if*), non aritmetici.

## Operatore Descrizione
 1. >  maggiore
 2. <  minore
 3. >= maggiore uguale
 4. <= minore uguale
 5. == uguale
 6. != diverso

### Nota: gli operatori di comparazione accettano come risultato solo il tipo boolean
