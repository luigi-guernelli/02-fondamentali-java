# Classe Main
La prima classe in Java è Main, che avvia il programma e mostra messaggi in output sulla console.
E' il punto di ingresso per ogni applicazione Java, dove definiamo istanze di oggetti metodi e attributi.

Nello specifico, usa *System.out.print()* per output senza a capo 
*System.out.println()* per output con a capo.

### Esempi nel codice
- *println*: Stampa "Benvenuto..." va a capo.
- *print*: Concatena frasi sulla stessa riga.
- *\n*: Forza un newline manuale.

### Commenti nel codice:
- *//*: Per una riga.
- */* */*: Per più righe.
