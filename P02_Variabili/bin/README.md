# Classe Main (Variabili)
Questa classe introduce le variabili in Java come contenitori per dati.

### Esempi nel codice
- **Dichiarazione**: *int x*; (definisce il tipo).
- **Assegnazione**: *x = 45* (impostazione del valore).
- **Inizializzazione**: *int y = 23* (dichiarazione + assegnazione).

### Tipi usati
- *int*: Numeri es.(45; -10; 0; riassegnamibili);
- *double*: Decimali es.(12.2);
- *String*: Testo es.("Ciao").

### Nomenclatura: Usa lettere minuscole, underscore per parole multiple es.( nome_utente ).

**Output:** Concatena variabili con *+* in *System.out.println()*.
