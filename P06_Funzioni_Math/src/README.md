# Classe Main (Math)
Questa classe USA *Math* utility statica per operazioni matematiche comuni.


### Operatore   ||    Descrizione
- **(abs)**:              Valore assoluto toglie segno
- **(ma(a, b))**:        Max tra due valori
- **(pow(base, exp))**:   Potenza (base^exp)
- **(sqrt(n))**:          Radice quadrata
