# Classe Main (Array Oggetti)
Questa classe esplora array di oggetti: contengono reference (non copie) a istanze.
**Zero-based:** indici 0 a *lenght-1*: accesso array[i] restituisce oggetto (usa toString() per print).
**Vuoto:** Inizializzazione null, popola con assegnazioni; pieno: Diretto con *{}*.

### Demo
- **Array vuoto**: Prima (length: 3); dopo assegnazione → Stampa "Elemento [0]: Giovanni Rossi (nato nel 1988, età 37)" ... <br><br>

- **Array pieno**: Length: 3; Stampa stessi elementi.


### Output
=== ARRAY VUOTO (prima dell'assegnazione) === <br>
Lunghezza: 3 <br>
=== ARRAY VUOTO (dopo assegnazione) === <br>
Elemento [0]: Giovanni Rossi (nato nel 1988, età 37) <br>
Elemento [1]: Mario Verdi (nato nel 1987, età 38) <br>
Elemento [2]: Maria Luisa (nato nel 2000, età 25) <br>
=== ARRAY PIENO === <br>
Lunghezza: 3 <br>
Elemento [0]: Giovanni Rossi (nato nel 1988, età 37) <br>
... (stessi) <br>
