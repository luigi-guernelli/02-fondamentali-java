# Classe Main (Tipi Primitivi)
Questa classe esplora i tipi primitivi (dati base) vs reference (oggetti).

- *Primitivi* inizializzati con valore di default es. int = 0; reference con null.

- *Reference* hanno metodi extre es.(*String.toUpperCase()*)

### Tipo,Descrizione,Range/Dimensione,Esempio
boolean = Vero/falso,1 bit, true
byte    = Intero piccolo, -128 a 127 (1 byte),127
short   = Intero medio, -32k a 32k (2 byte),-23456
int     = Intero standard, -2B a 2B (4 byte),2_000_000_000
long    = Intero grande,-9E18 a 9E18 (8byte),200_000_000_000_000_000L
float = Decimale corto, ~6-7 cifre (4 byte),1.23456789F
double = Decimale lungo, ~15 cifre (8 byte),1.23456789
char = Carattere, Unicode (2 byte),'A'
String (reference) = Testo,Variabile,"""Tipi primitivi e reference"""
