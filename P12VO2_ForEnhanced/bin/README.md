Classe Main per for enhanced (foreach) su array

# Classe Main (For Enhanced)
Questa classe dimostra il ciclo *for enhanced* o (*foreach*), per iterare array/collezioni direttamente sui valori, senza gestire indici.

# Struttura
*for (int numero : numeri) {...}* - *numero* prende ogni elemento di numeri.

# Vantaggi
Codice semplice, sicuro, ideale per latture sequanziali.

## Esempio con array
*int[ ] numeri = {10, 20, 30, 40, 50}*:

- **Stampa**: "Elemento array: 10" poi (20, 30, 40, 50).
Output finale:"Iterazione completa (5 elementi)".
