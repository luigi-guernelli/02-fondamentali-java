# Classe Main (Operatori Aritmetici)
Questa classe dimostra operatori aritmetici base.

### Operatore Descrizione Esempio Risultato.
- (+ addizione)           10 + 5 = 15;
- (- sottrazione)         10 - 5 = 5
- (* moltiplicazione)     10 x 5 = 50
- (/ divisione)           10 / 5 = 2.0
- (% Modulo (resto))      10 % 5 = 0
