# Classe Main (Break e Continue)
Questa classe dimostra *break* e *continue* per controllare flusso in cicli (for/while).
Automatizzano salti o uscite prematuri, migliorando efficienza.


## Istruzione,Descrizione,Effetto su for (i=0 to 14),Esempio
1. *continue* = "Salta resto iterazione corrente, va alla prossima",
"Stampa 0-3, salta 4, continua 5-14",if (i==4) continue; → Ignora stampa i=4

2. *break* = Esce immediatamente dal ciclo,"Stampa 0-3, ferma a 4 (non esegue 5+)",if (i==4) break; → Termina loop
